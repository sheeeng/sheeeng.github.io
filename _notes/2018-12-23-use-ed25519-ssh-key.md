- Read [Risan's post][risans-post] about [Ed25519][ed25519-homepage].

- Check current available SSH keys.

```bash
#!/usr/bin/env bash

for key in ~/.ssh/id_*; do
    ssh-keygen -l -f "${key}"
done | uniq
```

- Generate the [Ed25519][ed25519-homepage] SSH keys.

```bash
ssh-keygen \
    -o -a 100 \
    -t ed25519 \
    -f ~/.ssh/id_ed25519 \
    -C "username@hostname"
```
